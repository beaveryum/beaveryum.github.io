전체 검색에 단어가 짤리는 현상를 sir에 문의 했는데 답변이 이렇다. 



검색단어를 늘릴려면

bbs/search.php 파일의 대략 63 번째줄의 코드

$s = array_slice($s, 0, 2);

를

$s = array_slice($s, 0, 3);

으로 수정하면 3개 단어로 늘릴수 있습니다.

게시판이 많으면 검색량이 늘어나서 sql 부하가 일어날수 있습니다.
sql 검색량이 과도하게 일어나는 것은 취약점으로 여겨질수 있기 때문에, 현재 2개까지 단어를 허용하게 해 놓았습니다.
