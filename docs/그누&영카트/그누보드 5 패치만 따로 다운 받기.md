커맨드 창을 열기!!

```bash
cd /해당경로
curl -LO "https://github.com/gnuboard/gnuboard5/releases/download/v5.4.15/gnuboard5.4.15.patch.zip"
&& unzip gnuboard5.4.15.patch.zip
```

 
위에 버전을 확인하면 릴리즈를 확인해야한다.

https://github.com/gnuboard/gnuboard5/releases
